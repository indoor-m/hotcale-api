## スタートガイド - Getting Started

1. .env.exampleをコピーしてオリジンの修正が必要
```
ALLOW_ORIGINS=["https://localhost:3000",""https://localhost:3001"]
```

2. docker-compose で立ち上げる
```
# Docker の立ち上げ
docker-compose up -d --build
```
